# Disable Google AI Mode

Chrome extension that removes AI Mode from Google search pages and homepage.

## Features

- Removes AI Mode button from Google homepage
- Removes AI Mode tab from search results
- Blocks direct AI Mode URLs
- Lightweight and privacy-friendly

## Installation

1. Download the extension
2. Open Chrome
3. Go to chrome://extensions
4. Enable Developer Mode
5. Click Load Unpacked
6. Select the extension folder

## Privacy

This extension does not collect or store any user data.
