# Privacy Policy

Disable Google AI Mode is a browser extension that modifies the Google Search interface to remove AI Mode elements and block Gemini AI pages.

## Data Collection

This extension does not collect, store, or transmit any personal or sensitive user data.

## Browsing Data

The extension only modifies the visual interface of Google pages locally in the user's browser. It does not track browsing activity, search queries, or user behavior.

## Data Sharing

No user data is shared with any third parties.

## Remote Code

All code used by this extension is packaged within the extension itself. No remote scripts or external code are executed.

## Contact

For questions regarding this privacy policy, please contact the developer through the extension repository.
